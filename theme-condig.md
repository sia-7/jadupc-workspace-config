## Theme Config:

Theme link : https://github.com/vinceliuice/Orchis-theme

### Download 

```sh
git clone https://github.com/vinceliuice/Orchis-theme.git
```

### Install

```sh
cd Orchis-theme && ./install.sh --size standard  --tweaks black macos primary --round 3px
```
